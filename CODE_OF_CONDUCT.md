# Code of Conduct

This project adopts the [Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full canonical text is at the link above. By participating in this project — through issues, pull requests, discussions, or any other channel — you agree to abide by it.

---

## Summary

Participants are expected to behave with respect, openness, and empathy. Behaviour that creates a hostile or unwelcoming environment for others is not tolerated.

The Contributor Covenant defines specific examples of acceptable and unacceptable behaviour, the responsibilities of project maintainers, the scope of enforcement, and the process for reporting concerns. Read the full text at the canonical URL above.

---

## Reporting

To report behaviour that may violate this code, contact the project maintainer directly:

**Simon Conway · simon.conway@condaal.com**

Reports are handled confidentially. Action will be taken in line with the enforcement guidelines defined in Contributor Covenant 2.1, section "Enforcement Guidelines."
