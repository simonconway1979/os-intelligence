# Acme Corp Example Project

**Fully synthetic.** None of the people, companies, decisions, or events in this folder are real.

A populated OS-Intelligence project after one day of use. Simon Conway has just joined Acme Corp (a fictional UK B2B recruitment SaaS) as Head of Product — AI. Over four days he's done a six-meeting listening tour, read 21 source documents, scrolled four weeks of leadership Slack, and run the OS-Intelligence skill stack against the lot. This folder is the result.

## Recreate this yourself

The 21 raw docs, 6 transcripts, and 1 chat in this folder are the source material. To recreate the output, take those raw files into a fresh project and run `/ctx-doc`, `/ctx-transcript`, `/ctx-chat`, `/ctx-synthesise`.
